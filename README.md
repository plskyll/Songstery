# Songstery — Вебсайт для пошуку музичних рекомендацій до книг

## Опис проєкту

Songstery — це вебплатформа, що поєднує літературу та музику. Користувачі можуть шукати книги за назвою, автором чи обкладинкою, переглядати розділи без тексту, а також знаходити або додавати музичні композиції, які асоціюються з певними розділами. Проєкт створений для того, щоб допомогти читачам глибше зануритися в атмосферу книги через музику та створити спільний простір емоційного обміну.

## Мета проєкту

- Створити вебсайт, який об'єднує читачів та музичних ентузіастів
- Реалізувати інтерактивну систему пошуку книг і рекомендацій музики
- Забезпечити можливість користувачам взаємодіяти — додавати власні композиції, ставити лайки та формувати емоційні добірки

## Основні функції

- Пошук книг за назвою, автором або обкладинкою
- Перегляд інформації про книгу та її розділи
- Перегляд і додавання музичних рекомендацій до кожного розділу
- Оцінювання (лайки) музичних композицій інших користувачів
- Реєстрація та авторизація користувачів

## Інструкція запуску

### Передумови

- Python 3.8 або вище
- pip (менеджер пакетів Python)
- Git

### Крок 1: Клонування репозиторію

```bash
git clone https://github.com/your-username/songstery.git
cd songstery
```

### Крок 2: Створення віртуального середовища

```bash
python -m venv venv
```

### Крок 3: Активація віртуального середовища

**Windows:**
```bash
venv\Scripts\activate
```

**Linux/macOS:**
```bash
source venv/bin/activate
```

### Крок 4: Встановлення залежностей

```bash
pip install -r requirements.txt
```

### Крок 5: Виконання міграцій

```bash
python manage.py migrate
```

### Крок 6: Створення суперкористувача (опціонально)

```bash
python manage.py createsuperuser
```

### Крок 7: Запуск сервера

```bash
python manage.py runserver
```

### Крок 8: Відкрити у браузері

Перейдіть за адресою: [http://127.0.0.1:8000/](http://127.0.0.1:8000/)

Адміністративна панель: [http://127.0.0.1:8000/admin/](http://127.0.0.1:8000/admin/)
